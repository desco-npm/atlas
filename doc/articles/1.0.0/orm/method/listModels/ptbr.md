# listModels()

O método `listModels()` do [ORM](#orm) retorna uma lista com todos os [Models](#orm.model).

## Parâmetros

* Nenhum

## Retorno

[Array](https://developer.mozilla.org/pt-br/docs/Web/JavaScript/Reference/Global_Objects/Array) com todos os [Models](#orm.model).
